Allgemeines

  • 2 mal im Jahr wird das Kurzprogramm von der Bildung auf die Pfiffikus Plattform vom BIFO gestellt
  • Dazu wird ein Export einer csv Datei aus SemiQ erstellt
  • Diese Daten werden etwas nachbearbeitet
    • Titel wird angepasst
  • Dieses CSV wird dann per SFTP auf den Pfiffikus Server hochgeladen
